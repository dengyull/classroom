Instructions

CSI 2372

Lab Assignment 1

In this lab assignment, you will write a small program that reads in scores of a sports team and calculates some numbers. The scores should be entered on the console integer, e.g., "5:2 Toronto", meaning your team scored 5 goals while the opponents (Toronto) scored 2. The program is to be flexible on how many scores will be enteres and hence you must use a std::vector to store the scores and the opponents. The input will terminate when the user enters a negative number. At this point you will need to analyze the scores. You will need to calculate some statistics (see below) and display the results on the console.

  1. Create the source file scores.cpp. For this first lab, this is all we will use.

  2. Include the standard header file for vectors vector and for strings string.

  3. In the main routine write a loop that reads in numbers and text and stores each score and opponents in a vector. If the score is negative, the input must stop. The negative score must not be stored.

  4. In the main routine write a loop that calculates the number of wins, draws and losses of your team, the total goals scored and received. You will also need to calculate the average number of goals scored per game. Further, you must display the highest win and largest loss including the respective name of the opponents (break ties in a reasonable way). Finally repeat all the results.

  5. Display your results reasonably formated to the console.



An example run of your program will look as follows:

5:2 Toronto
7:4 Montreal
1:4 Winnipeg
3:2 LA
-1
Wins/Draws/Losses: 3:0:1
Goals: 16:12
Average number of goals scored: 4.0
Highest win: 7:4 Montreal
Largest loss: 1:4 Winnipeg
Games:
5:2 Toronto
7:4 Montreal
1:4 Winnipeg
3:2 LA

You must hand-in one file exactly named: scores.zip containing scores.cpp. Do not hand-in any other files (especially no Visual studio project files) or use any other form of archive (no rar). No word files or pdfs (or similar word processing files) will be accepted and if submitted will result in an automatic 0.
